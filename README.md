🍽️ Eat-N-Split React App

A simple React application that helps users split bills with friends and track who owes whom.
This project was built as a practice project to strengthen core React concepts such as state management, component composition, and conditional rendering.

🚀 Features

👥 Add Friends

Add new friends with a name and avatar.

📋 Friends List

Display all friends and their current balance.

💸 Split Bills

Enter the total bill, your expense, and who paid the bill.

📊 Balance Tracking

The app automatically calculates and updates who owes money.

⚡ Instant UI Updates

React state updates trigger re-rendering without page refresh.

🛠️ Built With

React.js

JavaScript (ES6+)

CSS

🧠 React Concepts Practiced

This project focuses on practicing important React fundamentals:

Component-based architecture

Props and component communication

State management with useState

Conditional rendering

Handling controlled form inputs

Updating arrays and objects in state

⚙️ Installation

Clone the repository:

git clone https://github.com/20Hanan/eat-n-split-react-project.git

Navigate to the project folder:

cd eat-n-split-react-project

Install dependencies:

npm install

Run the development server:

npm start

The app will run on:

http://localhost:3000
🎯 Purpose of the Project

The goal of this project is to practice essential React development skills and understand how data flows between components in a small interactive application.
