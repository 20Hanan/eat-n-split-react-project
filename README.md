# 🍽️ Eat-N-Split React App

A simple React application that helps users split bills with friends and track who owes money.
The app allows you to add friends, select who paid the bill, and automatically calculate balances between users.

This project was built to practice core **React fundamentals** such as state management, component structure, conditional rendering, and handling form inputs.

---

## 🚀 Features

* 👥 **Add Friends**
  Users can add new friends with a name and avatar.

* 📋 **Friends List**
  Displays all friends and their current balance status.

* 💸 **Split Bills**
  Enter the bill amount, your share, and choose who paid.

* 📊 **Automatic Balance Calculation**
  The app calculates who owes money and updates balances instantly.

* ⚡ **Dynamic UI Updates**
  React state updates allow the UI to update instantly without refreshing the page.
